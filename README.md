# FAQ Accordion

A responsive FAQ accordion component built using semantic HTML, modern CSS, and vanilla JavaScript. This project focuses on clean layout structure, interactive UI behavior, and responsive design.

## 🛠️ Built With
- HTML5 (Semantic structure)
- CSS3 (Flexbox, custom properties)
- JavaScript (DOM manipulation, event handling)

---

## ✨ Features
- Interactive accordion (expand/collapse answers)
- Only one section open at a time
- Dynamic icon switching (+ / −)
- Responsive layout for different screen sizes
- Clean and minimal UI

---

## 📚 What I Learned
- Handling user interactions with JavaScript
- Using `classList` for state management
- Structuring reusable and clean HTML
- Managing spacing and layout with Flexbox
- Debugging CSS issues like unwanted margins and padding

---

## ⚡ Challenges Faced
- Controlling spacing without breaking layout
- Handling default browser styles (like margins)
- Implementing toggle logic while closing other items
- Structuring CSS selectors without affecting unintended elements
